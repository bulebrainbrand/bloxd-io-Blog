setPosition

      
      api.setPosition(entityId,x,y,z)
      api.setPosition(playerId,[x,y,z])

説明

引数

  • entityId - 移動させるエンティティのid
  • x,y,z　または　[x,y,z] - 移動先の座標

jsdoc

      
/**
 * Set position of a player / entity.
 *
 * @param {EntityId} entityId
 * @param {number | number[]} x - Can also be an array, in which case y and z shouldn't be passed
 * @param {number} [y]
 * @param {number} [z]
 * @returns {void}
 */
setPosition(entityId, x, y, z)
